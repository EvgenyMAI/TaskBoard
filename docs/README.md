# Структура папки `docs`

## Назначение

Папка содержит материалы документации проекта: архитектурные диаграммы, ER-диаграммы баз данных, вспомогательные скрипты и текст отчёта по ВКР.

## Содержимое

- `diagrams/architecture/` - архитектурная схема приложения:
  - `architecture.mmd` (исходник Mermaid),
  - `architecture.png` (сгенерированное изображение).
- `diagrams/er/` - ER-диаграммы, экспортированные из DataGrip:
  - `er-auth.png`,
  - `er-tasks.png`,
  - `er-analytics.png`.
- `scripts/` - скрипты генерации диаграмм.
- `api.http` - набор HTTP-запросов для ручной проверки API.
- `curl-checks.md` - CLI-чеклист для API и тестовых сценариев.
- `report-vkr.md` / `report-vkr.docx` - отчётные материалы ВКР.
